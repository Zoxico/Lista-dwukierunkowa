# Lista-dwukierunkowa
Implementacja listy dwukierunkowej.

FUNKCJONALNOŚCI ZAWARTE W PROJEKCIE:
- INSERT    - Wprowadzanie wartości
- DELETE    - Usuwanie wartości
- MAX       - Szukanie największej wartości
- MIN       - Szukanie najmniejszej wartości
- MEMBER    - Sprawdzanie czy dana wartość jest elementem listy
- IN ORDER  - Wyświetlanie posortowanych elementów
- SUCCESSOR - Szukanie poprzedniego elementu od podanego
